# DevTools Kit - 开发者在线工具箱

15+ 常用开发工具，纯浏览器端运行，数据不上传，安全隐私无忧。

## 工具列表

| 工具 | 说明 |
|------|------|
| JSON 格式化 | 格式化、校验、压缩 JSON 数据 |
| Base64 编解码 | Base64 编码与解码转换 |
| 时间戳转换 | Unix 时间戳与日期互转 |
| URL 编解码 | URL 编码与解码 |
| 二维码生成 | 文本/链接在线生成二维码 |
| 正则测试 | 实时正则表达式测试与替换 |
| 图片压缩 | 在线压缩 PNG/JPEG/WebP |
| 颜色转换 | HEX/RGB/HSL 互转 |
| UUID 生成 | 批量生成 UUID/GUID |
| 文本对比 | 双栏文本差异对比 |
| HTML 实体 | HTML 实体编解码 |
| Unicode 转换 | Unicode 与中文互转 |
| JWT 解码 | 在线解析 JWT Token |
| 哈希计算 | MD5/SHA1/SHA256 哈希 |
| Markdown 预览 | 实时 Markdown 编辑预览 |
| 字数统计 | 字符/单词/行数统计 |

## 技术栈

React 19 + TypeScript + Vite 8 + TailwindCSS 4

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```
